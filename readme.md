# How to run it

Run npm install and npm start from the backend folder in the terminal

Open `frontend/index.html` in a browser while the backend is running on port `8080`.

The backend exposes the API at `http://localhost:8080/professional`.

## Environment variables

Don't forget to create the .env file. If you are unsure of how to do this, watch the stretch solution video.
